--

### Get It! 

__Latest stable version for your dhis2 release__

> Use @<dhis2-release-version> to get the latest version applicable to that installation

For DHIS2 2.25
```
npm install --save d2-ui@25
```

For DHIS2 2.24
```
npm install --save d2-ui@24
```
